Final Year Project

Mobile Applicaton written in Flutter, with collaboration from Bank of America Merrill Lynch. The application tracks user counts in rooms by connecting their phones to Bluetooth beacons. On the app, one can see how many people are in each room, and if the room is over the max people limit.

Second Year Project

3-person group project that consisted of creating an online store selling PC components. The web page was created using Python Django/Wagtail CMS and designed using Bootstrap and CSS. Some functionality included a virtual shopping cart for storing items, a voucher code system for discounts, sign up and sign in system for creating and logging into accounts as well as 2 Factor Authentication (2FA) for extra security. Credit card payment system using Stripe and a live support chat using a WebSocket. The stores database was managed using SQLite.
